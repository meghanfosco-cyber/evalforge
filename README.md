# EvalForge

EvalForge is a polished React/Next.js prototype for AI evaluation and prompt QA workflows. It includes a landing page, a structured evaluation workspace, rubric presets, sample demos, and deterministic mock scoring.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included rubrics

- Text Evaluation
- Image Generation Comparison
- UI Screenshot Description
- Agent Tool-Use Review
